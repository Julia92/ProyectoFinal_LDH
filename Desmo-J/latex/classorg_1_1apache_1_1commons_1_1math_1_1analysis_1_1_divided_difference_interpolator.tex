\subsection{Detailed Description}
Implements the $<$a href=\char`\"{}
\char`\"{}{\tt http\-://mathworld.\-wolfram.\-com/\-Newtons\-Divided\-Difference\-Interpolation\-Formula.\-html}"$>$ Divided Difference Algorithm for interpolation of real univariate functions. For reference, see {\bfseries Introduction to Numerical Analysis}, I\-S\-B\-N 038795452\-X, chapter 2. 

The actual code of Neville's evalution is in \doxyref{Polynomial\-Function\-Lagrange\-Form}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_function_lagrange_form}, this class provides an easy-\/to-\/use interface to it.

Returns a copy of the divided difference array. 

The divided difference array is defined recursively by 
\begin{DoxyPre}
f[x0] = f(x0)
f[x0,x1,...,xk] = (f(x1,...,xk) - f(x0,...,x[k-1])) / (xk - x0)
\end{DoxyPre}


The computational complexity is O(\-N$^\wedge$2).

a[] and c[] are defined in the general formula of Newton form\-: p(x) = a[0] + a[1](x-\/c[0]) + a[2](x-\/c[0])(x-\/c[1]) + ... + a[n](x-\/c[0])(x-\/c[1])...(x-\/c[n-\/1])

When used for interpolation, the Newton form formula becomes p(x) = f[x0] + f[x0,x1](x-\/x0) + f[x0,x1,x2](x-\/x0)(x-\/x1) + ... + f[x0,x1,...,x[n-\/1]](x-\/x0)(x-\/x1)...(x-\/x[n-\/2]) Therefore, a[k] = f[x0,x1,...,xk], c[k] = x[k]. 

Note x[], y[], a[] have the same length but c[]'s size is one less.
