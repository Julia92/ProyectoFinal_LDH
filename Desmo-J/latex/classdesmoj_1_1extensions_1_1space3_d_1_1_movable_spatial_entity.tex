\subsection{Detailed Description}
The class represents the movable D\-E\-S\-M\-O-\/\-J Entities which should be visualized. Therefore it implements the \doxyref{Movable\-Spatial\-Object}{p.}{interfacedesmoj_1_1extensions_1_1space3_d_1_1_movable_spatial_object} interface for the handling with the 3\-D spatial data, the spatial movements and the parent-\/child-\/\-M\-S\-O concept.

There are different varieties of movement modes depends on the initialized parameters (O = initialized, X = not initialized(means invalid or null))\-:

case 1\-: duration O, max\-Speed O, acc \& dec O\-: The movement will be performed according to the parameters. If the given parameters can't be satisfied, an \doxyref{Illegal\-Move\-Exception}{p.}{classdesmoj_1_1extensions_1_1space3_d_1_1_illegal_move_exception} will be thrown.

case 2\-: duration X, max\-Speed O, acc \& dec O\-: The object will be moved to the destination with the fastest speed possible.

case 3\-: duration O, max\-Speed X, acc \& dec O\-: The object will move to the destination with the given acceleration and deceleration and try to reach the destination in time. If the given parameters can't be satisfied, an \doxyref{Illegal\-Move\-Exception}{p.}{classdesmoj_1_1extensions_1_1space3_d_1_1_illegal_move_exception} will be thrown.

case 4\-: duration X, max\-Speed X, acc \& dec O\-: The object will be accelerated and followed by a deceleration. There's no constant phase in the movement.

case 5\-: duration O, max\-Speed O, acc or dec X\-: The object will start with the speed needed to arrive the destination in the given time and break suddenly to zero at the end of the movement. The max\-Speed gives the speed limit of the movement. If the given parameters can't be satisfied, an \doxyref{Illegal\-Move\-Exception}{p.}{classdesmoj_1_1extensions_1_1space3_d_1_1_illegal_move_exception} will be thrown.

case 6\-: duration X, max\-Speed O, acc or dec X\-: The movement will start with the max\-Speed and stops the destination.

case 7\-: duration O, max\-Speed X, acc or dec X\-: The movement will start with the speed needed to arrive the destination with the duration given.
