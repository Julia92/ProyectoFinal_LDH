Returns an exact representation of the Binomial Coefficient, "<code>n choose k</code>", the number of k-element subsets that can be selected from an n-element set.

static double org.apache.commons.math.util.MathUtils.normalizeAngle ( double  a,
double  center 
)
static

Normalize an angle in a 2&pi wide interval around a center value.

This method has three main uses:

  • normalize an angle between 0 and 2π:
    a = MathUtils.normalizeAngle(a, Math.PI);
  • normalize an angle between -π and +π
    a = MathUtils.normalizeAngle(a, 0.0);
  • compute the angle between two defining angular positions:
    angle = MathUtils.normalizeAngle(end, start) - start;

Note that due to numerical accuracy and since π cannot be represented exactly, the result interval is closed, it cannot be half-closed as would be more satisfactory in a purely mathematical view.

Parameters
aangle to normalize
centercenter of the desired 2π interval for the result
Returns
a-2kπ with integer k and center-π <= a-2kπ <= center+π
Since
1.2
