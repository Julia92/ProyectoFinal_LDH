Detailed Description

EntityScheduleEntry is an entry of EntitySchedule. A EntityScheduleEntry is a time region (e.g. every workingday from 8:00 until 16:00) All EntityScheduleEntry of a EntitySchedule have the same timezone, set by EntitySchedule.
