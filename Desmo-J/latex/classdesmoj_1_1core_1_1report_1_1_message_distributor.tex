\subsection{Detailed Description}
The central object for distributing the messages generated by a simulation run. The \doxyref{Message\-Distributor}{p.}{classdesmoj_1_1core_1_1report_1_1_message_distributor} can receive messages and reporters and forwards them to the {\ttfamily \doxyref{Message\-Receiver}{p.}{interfacedesmoj_1_1core_1_1report_1_1_message_receiver}} objects registered at this \doxyref{Message\-Distributor}{p.}{classdesmoj_1_1core_1_1report_1_1_message_distributor}. When registering, the {\ttfamily \doxyref{Message\-Receiver}{p.}{interfacedesmoj_1_1core_1_1report_1_1_message_receiver}} has to pass a type to identify which type of messages they want to have forwarded. Note that multiple {\ttfamily \doxyref{Message\-Receiver}{p.}{interfacedesmoj_1_1core_1_1report_1_1_message_receiver}} s can be registered to get the same type of messages as well as a {\ttfamily \doxyref{Message\-Receiver}{p.}{interfacedesmoj_1_1core_1_1report_1_1_message_receiver}} can be registered with different types of messages, if it is capable of handling such. This enables a modeller i.\-e. to get the error messages displayed on screen additional to the file being stored on harddisk by default. This is also handy if a simulation should be run as an Applet thus having no or restricted disk access and using multiple scrollable windows instead.
