\subsection{Detailed Description}
\begin{DoxyRefDesc}{Deprecated}
\item[{\bf Deprecated}]Represents points in simulation time. Is used to indicate points in simulation time at which the state of the model changes. Each point in simulation time is represented by an individual object of this class and offers its own methods for arithmetic operations. Ensures that only valid points of time are generated.\end{DoxyRefDesc}
