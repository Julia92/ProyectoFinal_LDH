\subsection{Detailed Description}
An abstract implementation of a hash-\/based map that allows the entries to be removed by the garbage collector. 

This class implements all the features necessary for a subclass reference hash-\/based map. Key-\/value entries are stored in instances of the {\ttfamily Reference\-Entry} class which can be overridden and replaced. The iterators can similarly be replaced, without the need to replace the Key\-Set, Entry\-Set and Values view classes. 

Overridable methods are provided to change the default hashing behaviour, and to change how entries are added to and removed from the map. Hopefully, all you need for unusual subclasses is here. 

When you construct an {\ttfamily \doxyref{Abstract\-Reference\-Map}{p.}{classorg_1_1apache_1_1commons_1_1collections_1_1map_1_1_abstract_reference_map}}, you can specify what kind of references are used to store the map's keys and values. If non-\/hard references are used, then the garbage collector can remove mappings if a key or value becomes unreachable, or if the J\-V\-M's memory is running low. For information on how the different reference types behave, see \doxyref{Reference}{p.}{}. 

Different types of references can be specified for keys and values. The keys can be configured to be weak but the values hard, in which case this class will behave like a {\tt {\ttfamily Weak\-Hash\-Map}}. However, you can also specify hard keys and weak values, or any other combination. The default constructor uses hard keys and soft values, providing a memory-\/sensitive cache. 

This \doxyref{Map}{p.}{} implementation does {\itshape not} allow null elements. Attempting to add a null key or value to the map will raise a {\ttfamily Null\-Pointer\-Exception}. 

All the available iterators can be reset back to the start by casting to {\ttfamily \doxyref{Resettable\-Iterator}{p.}{interfaceorg_1_1apache_1_1commons_1_1collections_1_1_resettable_iterator}} and calling {\ttfamily reset()}. 

This implementation is not synchronized. You can use \doxyref{java.\-util.\-Collections\#synchronized\-Map}{p.}{} to provide synchronized access to a {\ttfamily \doxyref{Reference\-Map}{p.}{classorg_1_1apache_1_1commons_1_1collections_1_1map_1_1_reference_map}}.
