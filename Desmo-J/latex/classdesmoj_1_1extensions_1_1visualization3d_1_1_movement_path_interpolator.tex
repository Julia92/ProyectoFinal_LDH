\subsection{Detailed Description}
An interpolator which realize the 3-\/phase-\/translation. The translation is divided into 3 phases\-:
\begin{DoxyItemize}
\item The acceleration phase
\item The maximum speed phase
\item The deceleration phase The translation behavior is defined over the speed and duration of each phase. \begin{DoxyAuthor}{Author}
