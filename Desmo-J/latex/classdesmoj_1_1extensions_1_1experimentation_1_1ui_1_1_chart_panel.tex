\subsection{Detailed Description}
A G\-U\-I container (inherited from javax.\-swing.\-J\-Panel) which includes a header, legend items (if needed) and a special \doxyref{Chart}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_chart} object. All functionality of the \doxyref{Chart}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_chart} object which is needed by the user, is encapsulated in this class, so the user doesn{\ttfamily t need to have direct access to the \doxyref{Chart}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_chart} object, he doesn}t need to care about the \doxyref{Chart}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_chart} object at all.

This class additionally includes a method save\-As(filename) for saving the chart to an image file.

If you are a user of D\-E\-S\-M\-O-\/\-J you don`t need to create a object of this kind\-: Create a \doxyref{Histogram\-Plotter}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_histogram_plotter} or a \doxyref{Time\-Series\-Plotter}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_time_series_plotter} instead!

This class is needed by the graphical observers \doxyref{Histogram\-Plotter}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_histogram_plotter} and \doxyref{Time\-Series\-Plotter}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_time_series_plotter}. If you want to extend D\-E\-S\-M\-O-\/\-J you can create new plotter classes inherited from \doxyref{Graphical\-Observer}{p.}{classdesmoj_1_1extensions_1_1experimentation_1_1ui_1_1_graphical_observer} which contains this object as a private field \char`\"{}my\-G\-U\-I\char`\"{} and a method draw\-Chart(\-Graphics) for implementing the drawing of the data.
