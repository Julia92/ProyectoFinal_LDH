\subsection{Detailed Description}
A \doxyref{Truck}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_truck} is an {\ttfamily \doxyref{External\-Transporter}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_external_transporter}} which arrives at a container terminal to deliver and /or pick up one some containers (goods). It has a certain number of import/export containers that must have loaded/unloaded before it can leave a container terminal. The both numbers must be not negative. It also has a certain speed that he drives in a container terminal with. the speed must be not negative or zero. \doxyref{Truck}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_truck} is derived from External\-Trasnporter. Its {\ttfamily \doxyref{life\-Cycle()}{p.}{classdesmoj_1_1core_1_1simulator_1_1_sim_process_a38bb9fa8a520a8bd0bd26e74737fb3a3}} must be implemented by the user in order to specify the behavior of the \doxyref{Truck}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_truck}.
