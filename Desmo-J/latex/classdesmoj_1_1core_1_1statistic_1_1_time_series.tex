\subsection{Detailed Description}
The {\ttfamily \doxyref{Time\-Series}{p.}{classdesmoj_1_1core_1_1statistic_1_1_time_series}} class is recording data from a {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} object and saving it to a file, making data available to a Time\-Series\-Plotter or doing both. Either every time the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} object has changed or at every tick of the Sim\-Clock. See the automatic parameter in the constructor. If writing to a file is claimed, the first line of the file contains the name of the file. Then each line of the file contains a Sim\-Time value and the value of the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} object at that point of time in the simulation, separated by a \char`\"{};\char`\"{}. \par
 One can choose when the recording of the values will start and stop. See the constructor for further information. \par
 Reseting the {\ttfamily \doxyref{Time\-Series}{p.}{classdesmoj_1_1core_1_1statistic_1_1_time_series}} object will delete the contents of the file. \par
 This {\ttfamily \doxyref{Time\-Series}{p.}{classdesmoj_1_1core_1_1statistic_1_1_time_series}} object will be the observer of the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} object given in the constructor. When constructing a new {\ttfamily \doxyref{Time\-Series}{p.}{classdesmoj_1_1core_1_1statistic_1_1_time_series}} object it will be added to the list of observers of the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}}. That means this {\ttfamily \doxyref{Time\-Series}{p.}{classdesmoj_1_1core_1_1statistic_1_1_time_series}} object will be updated automatically every time the observed {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} has changed and has called its {\ttfamily notify\-Statistics()} method.

As separator between different values (e.\-g. Sim\-Time and value of the Valuesupplier) \char`\"{};\char`\"{} is used. See also File\-Output.
