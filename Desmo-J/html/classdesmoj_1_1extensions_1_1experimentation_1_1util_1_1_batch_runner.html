Detailed Description

A class to run a batch of simulation experiments from the console. The batches must be specified in an XML file. The XML file must start with a <batch>tag. In the top level tag it is possible to specify model and experiment settings valid for every run using the <model>and <exp>tag known from the Run class. Furthermore an arbitrary number of runs (specified by a <run>tag) containing the variable settings can be embedded.
