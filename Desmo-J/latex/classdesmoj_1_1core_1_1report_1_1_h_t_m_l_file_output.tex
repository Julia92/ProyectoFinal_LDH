\subsection{Detailed Description}
Extends the Std\-File\-Out writer by the ability to create H\-T\-M\-L pages displaying the given text in table format. Using the methods {\ttfamily void \doxyref{open\-Table(\-String name)}{p.}{classdesmoj_1_1core_1_1report_1_1_h_t_m_l_file_output_a9ec1f6f2f9b57c0c6308cf1080ac6dcc}} and {\ttfamily void \doxyref{close\-Table()}{p.}{classdesmoj_1_1core_1_1report_1_1_h_t_m_l_file_output_accaf18acb8b002d46391e07bf0ad3890}} multiple distinct H\-T\-M\-L tables with individual headings givne as parameter can be written into one file.
