\subsection{Detailed Description}
This constant \char`\"{}pseudo\char`\"{}-\/distribution returns a single constant predefined value of type Double. This \char`\"{}distribution\char`\"{} is most useful for testing purposes. The value to be returned can be specified at construction time.
