void desmoj.extensions.experimentation.ui.TimeSeriesPlotter.updatePlotter ( )

This method updates the chart. The chart will only be repainted if a new scaling is required. If no rescaling is required, then only the new values are painted. Of course this is much faster.

Because the TimeSeriesPlotter has no bench marks for scaling at the beginning, you have to call this method at least one time, when the upper and lower limits are known.
