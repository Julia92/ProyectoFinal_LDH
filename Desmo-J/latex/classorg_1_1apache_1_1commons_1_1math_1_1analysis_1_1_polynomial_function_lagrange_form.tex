\subsection{Detailed Description}
Implements the representation of a real polynomial function in {\tt Lagrange Form}. For reference, see {\bfseries Introduction to Numerical Analysis}, I\-S\-B\-N 038795452\-X, chapter 2. 

The approximated function should be smooth enough for Lagrange polynomial to work well. Otherwise, consider using splines instead.
