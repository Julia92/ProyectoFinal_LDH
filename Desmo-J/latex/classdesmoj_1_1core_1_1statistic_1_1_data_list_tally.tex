\subsection{Detailed Description}
The {\ttfamily \doxyref{Data\-List\-Tally}{p.}{classdesmoj_1_1core_1_1statistic_1_1_data_list_tally}} class is providing a statistic analysis about one value. The mean value and the standard deviation is calculated on basis of the total number of observations. All observed values are stored in a list to be able to calculate quartiles or histograms.
