\subsection{Detailed Description}
A \doxyref{Simple\-Track}{p.}{classdesmoj_1_1extensions_1_1space3_d_1_1_simple_track} contains a start position and a destiny position. The route between them are modeled through way points. Each point (incl. the start point) is connected with the successor point through a straight line. By following the connected way points from the start point the destiny will be reached at the end.
