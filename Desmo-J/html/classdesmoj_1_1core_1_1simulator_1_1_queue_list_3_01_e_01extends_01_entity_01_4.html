Detailed Description

Is the abstract superclass for all the classes implementing different queueing strategies for a waiting-queue. It provides all the basic methods for inserting objects in a queue, retrieving objects from a queue and getting basic informations about the queue. It is used in many kinds of queue implementations where collective functionalities are implemented by QueueListStandard and are specified e.g. in QueueListFifo or QueueListLifo.
