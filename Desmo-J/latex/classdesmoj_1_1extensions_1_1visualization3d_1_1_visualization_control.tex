\subsection{Detailed Description}
This class provides the central control unit for the spatial visualization. It's the connector between the Visualevent-\/\-Producer and the Visual\-Mudules. Each \doxyref{Visual\-Module}{p.}{interfacedesmoj_1_1extensions_1_1visualization3d_1_1_visual_module} is a different visualization implementation. The \doxyref{Visual\-Module}{p.}{interfacedesmoj_1_1extensions_1_1visualization3d_1_1_visual_module} to be used should be registered in the Visualization\-Controler. The controller will forward the events to each registered module.
