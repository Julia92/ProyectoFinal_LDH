\subsection{Detailed Description}
Implements the {\tt Neville's Algorithm} for interpolation of real univariate functions. For reference, see {\bfseries Introduction to Numerical Analysis}, I\-S\-B\-N 038795452\-X, chapter 2. 

The actual code of Neville's evalution is in \doxyref{Polynomial\-Function\-Lagrange\-Form}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_function_lagrange_form}, this class provides an easy-\/to-\/use interface to it.
