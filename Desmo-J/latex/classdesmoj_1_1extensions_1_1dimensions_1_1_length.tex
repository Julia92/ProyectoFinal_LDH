\subsection{Detailed Description}
This class represents lengths. As lengths are quantities each length consists of a numerical value and a unit. Only the length units specified in the \doxyref{Units}{p.}{interfacedesmoj_1_1core_1_1simulator_1_1_units}interface are supported. The value may not be negative.
