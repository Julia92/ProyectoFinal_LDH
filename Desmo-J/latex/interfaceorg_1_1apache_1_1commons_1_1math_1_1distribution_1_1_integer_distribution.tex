For this distribution, X, this method returns the largest x such that P(X {$\le$} x) $<$= p. 

Note that this definition implies\-: 
\begin{DoxyItemize}
\item If there is a minimum value, {\ttfamily m}, with postive probablility under (the density of) X, then {\ttfamily m -\/ 1} is returned by {\ttfamily inverse\-Cumulative\-Probability(0).} If there is no such value {\ttfamily m, Integer.\-M\-I\-N\-\_\-\-V\-A\-L\-U\-E} is returned. 
\item If there is a maximum value, {\ttfamily M}, such that P(X {$\le$} M) =1, then {\ttfamily M} is returned by {\ttfamily inverse\-Cumulative\-Probability(1).} If there is no such value, {\ttfamily M, Integer.\-M\-A\-X\-\_\-\-V\-A\-L\-U\-E} is returned.
\end{DoxyItemize}
