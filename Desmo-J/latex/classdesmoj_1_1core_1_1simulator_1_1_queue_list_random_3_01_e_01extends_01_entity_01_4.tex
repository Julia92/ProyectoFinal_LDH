\subsection{Detailed Description}
Contains the implementation with the java.\-util.\-Linked\-List to represent queueing functionality. The entities are queued in random order. The statistic data of the queue will be stored in a {\ttfamily \doxyref{Queue\-Based}{p.}{classdesmoj_1_1core_1_1simulator_1_1_queue_based}} object. The {\ttfamily Queue\-List\-Standard\-Fifo} has a reference to its {\ttfamily \doxyref{Queue\-Based}{p.}{classdesmoj_1_1core_1_1simulator_1_1_queue_based}} object. This class needs a reference to a subclass of \doxyref{Queue\-Based}{p.}{classdesmoj_1_1core_1_1simulator_1_1_queue_based} to update the queue statistics. It is used in many kinds of queue implementations i.\-e. in classes {\ttfamily Queue} and {\ttfamily Process\-Queue}.
