\begin{DoxyItemize}
\item {\ttfamily  lower\-Bound $<$= a $<$ initial $<$ b $<$= upper\-Bound}  
\item {\ttfamily  f(a) $\ast$ f(b) $<$ 0 } 
\end{DoxyItemize}If f is continuous on {\ttfamily [a,b],} this means that {\ttfamily a} and {\ttfamily b} bracket a root of f. 

The algorithm starts by setting {\ttfamily a \-:= initial -\/1; b \-:= initial +1,} examines the value of the function at {\ttfamily a} and {\ttfamily b} and keeps moving the endpoints out by one unit each time through a loop that terminates when one of the following happens\-: 
\begin{DoxyItemize}
\item {\ttfamily  f(a) $\ast$ f(b) $<$ 0 } -- success! 
\item {\ttfamily  a = lower } and {\ttfamily  b = upper} -- \doxyref{Convergence\-Exception}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1_convergence_exception}  
\item {\ttfamily  Integer.\-M\-A\-X\-\_\-\-V\-A\-L\-U\-E} iterations elapse -- \doxyref{Convergence\-Exception}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1_convergence_exception}  
\end{DoxyItemize}

{\bfseries Note\-: } this method can take {\ttfamily Integer.\-M\-A\-X\-\_\-\-V\-A\-L\-U\-E} iterations to throw a {\ttfamily \doxyref{Convergence\-Exception}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1_convergence_exception}.} Unless you are confident that there is a root between {\ttfamily lower\-Bound} and {\ttfamily upper\-Bound} near {\ttfamily initial,} it is better to use \doxyref{bracket(\-Univariate\-Real\-Function, double, double, double, int)}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_univariate_real_solver_utils_ab0760978d4f2b08eaf79548d92949580}, explicitly specifying the maximum number of iterations.
