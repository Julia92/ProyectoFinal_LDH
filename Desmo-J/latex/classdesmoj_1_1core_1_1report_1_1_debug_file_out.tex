\subsection{Detailed Description}
Debug\-Out is used to create a file to write the Debug\-Notes to. It receives a Debug\-Notes and divides its individual text information up to be displayed in H\-T\-M\-L format on disc in the user's local directory. The messages are displayed in a tabular design with columns for\-: 
\begin{DoxyItemize}
\item The point of simulation time the \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} was created 
\item The Model\-Component's name responsible for sending the \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} 
\item The textual description of the Model\-Component's current state 
\end{DoxyItemize}Note that if the framework is used as an Applet in webpages there is no access to the disc and an alternative output on screen or into a graphics window must be registered at the Experiment's Message\-Manager for this type of messages. Errors affecting the java runtime are always displayed on the system's standard output Print\-Stream.
