\subsection{Detailed Description}
Computes a natural (also known as \char`\"{}free\char`\"{}, \char`\"{}unclamped\char`\"{}) cubic spline interpolation for the data set. 

The \doxyref{interpolate(double[$\,$], double[$\,$])}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_spline_interpolator_abea10da05b93c933e3904121f25b9b0f} method returns a \doxyref{Polynomial\-Spline\-Function}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_spline_function} consisting of n cubic polynomials, defined over the subintervals determined by the x values, x[0] $<$ x[i] ... $<$ x[n]. The x values are referred to as \char`\"{}knot points.\char`\"{}

The value of the \doxyref{Polynomial\-Spline\-Function}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_spline_function} at a point x that is greater than or equal to the smallest knot point and strictly less than the largest knot point is computed by finding the subinterval to which x belongs and computing the value of the corresponding polynomial at {\ttfamily x -\/ x[i] } where {\ttfamily i} is the index of the subinterval. See \doxyref{Polynomial\-Spline\-Function}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_spline_function} for more details. 

The interpolating polynomials satisfy\-: 
\begin{DoxyEnumerate}
\item The value of the \doxyref{Polynomial\-Spline\-Function}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_spline_function} at each of the input x values equals the corresponding y value. 
\item Adjacent polynomials are equal through two derivatives at the knot points (i.\-e., adjacent polynomials \char`\"{}match up\char`\"{} at the knot points, as do their first and second derivatives). 
\end{DoxyEnumerate}

The cubic spline interpolation algorithm implemented is as described in R.\-L. Burden, J.\-D. Faires, $<$u$>$Numerical Analysis$<$/u$>$, 4th Ed., 1989, P\-W\-S-\/\-Kent, I\-S\-B\-N 0-\/53491-\/585-\/\-X, pp 126-\/131.
