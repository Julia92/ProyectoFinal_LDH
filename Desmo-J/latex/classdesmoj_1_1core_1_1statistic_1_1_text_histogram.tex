\subsection{Detailed Description}
The {\ttfamily \doxyref{Text\-Histogram}{p.}{classdesmoj_1_1core_1_1statistic_1_1_text_histogram}} class is providing a statistic analysis about text values. Given Strings are counted and the report will show the amount of every given text. If the given parameter is a null pointer it will not be counted.
