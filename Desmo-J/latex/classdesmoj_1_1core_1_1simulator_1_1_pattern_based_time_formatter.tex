\subsection{Detailed Description}
Use this class to format \doxyref{Time\-Instant}{p.}{classdesmoj_1_1core_1_1simulator_1_1_time_instant} objects using a java.\-text.\-Date\-Format object. The \doxyref{Pattern\-Based\-Time\-Formatter}{p.}{classdesmoj_1_1core_1_1simulator_1_1_pattern_based_time_formatter} uses a \doxyref{Multi\-Unit\-Time\-Formatter}{p.}{classdesmoj_1_1core_1_1simulator_1_1_multi_unit_time_formatter} to format Time\-Spans.
