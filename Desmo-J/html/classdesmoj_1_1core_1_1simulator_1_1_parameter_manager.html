Detailed Description

Class to manage parameters. Parameters are values, which affect a simulation- model or an experiment. Model-parameters are model's constants. Experiment- parameters are experiment-constants which can be used in the experiment's model.

This class is accessible from Model through the ModelParameterManager-interface to restrict the accessible methods from a model's view. It is accessible from Experiment through the ExperimentParameterManager-interface to restrict the accessible methods from a experiment's view.
