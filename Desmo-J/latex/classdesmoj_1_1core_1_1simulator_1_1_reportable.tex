This method provides a reporter to generate report data about this reportable. In this general implementation, a {\ttfamily Standard\-Reporter} is returned. Lacking further knowledge about a specific reportable, the {\ttfamily Standard\-Reporter} is only able to read out the reportable's name, number of observations and time of last reset.\par


Thus, it is highly recommended that subclasses override this method to return an appropriate default reporter for the specific reportable object. All reportable objects in D\-E\-S\-M\-O-\/\-J like {\ttfamily Accumulate}, {\ttfamily Tally}, {\ttfamily Count}, {\ttfamily Queue}, and many others provide such reporters. \par


If you want to extend the framework with new reportable objects, provide a suitable reporter as well and override this method to return an instance of the reporter. \par


If you just need to replace the reporter used by an existing reportable (while not requiring any changes to the reportable itself), there is no need to create a subclass of the reportable\-: Instead of overriding this method, just call {\ttfamily \doxyref{set\-Reporter()}{p.}{classdesmoj_1_1core_1_1simulator_1_1_reportable_a4db11882fd14bd226bc238253b0a0395}} to have this reportable use a custom reporter instead of the default reporter.
