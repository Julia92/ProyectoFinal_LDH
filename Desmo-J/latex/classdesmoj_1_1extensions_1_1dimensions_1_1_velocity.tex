\subsection{Detailed Description}
This class represents velocities (speeds). Each velocity consists of a numerical value and a unit. Only the speed units specified in the \doxyref{Units}{p.}{interfacedesmoj_1_1core_1_1simulator_1_1_units}interface are supported. The value may not be negative.
