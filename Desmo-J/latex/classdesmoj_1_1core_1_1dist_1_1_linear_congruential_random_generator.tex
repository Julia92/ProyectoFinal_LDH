\subsection{Detailed Description}
Linear congruential random generator for uniformly distributed pseudo random numbers configured such that a stream of [0,1] double values is produced. Implements the {\ttfamily desmoj.\-dist.\-Uniform\-Random\-Generator} interface. All {\ttfamily Distributions} in this package use this random generator by default. The implementation is based on the Java A\-P\-I {\ttfamily java.\-util.\-Random} class' random generator. The Java A\-P\-I Random class uses a 48-\/bit seed as input to the linear congruential formula. (See Donald Knuth, The Art of Computer Programming, Volume 2, Section 3.\-2.\-1.)
