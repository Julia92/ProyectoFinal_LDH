\subsection{Detailed Description}
Provides the class for user defined events to change the model's state at a distinct point in simulation time.

For events specifically changing the state of up to three entities, refer to {\ttfamily Event}, {\ttfamily Event\-Of2\-Entities} and {\ttfamily Event\-Of3\-Entities}. Events not associated to a specific entity are called external events as they are are typically used for external influences, e.\-g. arrivals from outside or incidents.

Derive from this class to design special external events for a model. To use external events, always create a new object of this class.
