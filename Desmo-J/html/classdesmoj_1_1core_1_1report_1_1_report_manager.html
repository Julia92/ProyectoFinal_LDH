void desmoj.core.report.ReportManager.deRegister ( Reporter  rep)

De-registers the given reporter from the experiment's reportmanager. The reporter will be removed from the list of current available reporters and thus will not produce output whenever a report has to be produced. When de-registering, the order according to group-ID is preserved. If an invalid parameter is given (i.e. a null reference) this method simply returns
