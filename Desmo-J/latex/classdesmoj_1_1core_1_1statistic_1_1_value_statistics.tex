\subsection{Detailed Description}
The {\ttfamily \doxyref{Value\-Statistics}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_statistics}} class is the super class for all the classes providing a statistic analysis about one value (e.\-g. minimum and maximum values). Derived classes are\-: \doxyref{Tally}{p.}{classdesmoj_1_1core_1_1statistic_1_1_tally}, \doxyref{Accumulate}{p.}{classdesmoj_1_1core_1_1statistic_1_1_accumulate} and \doxyref{Histogram}{p.}{classdesmoj_1_1core_1_1statistic_1_1_histogram} (because it is derived from \doxyref{Tally}{p.}{classdesmoj_1_1core_1_1statistic_1_1_tally}).
