\subsection{Detailed Description}
The \doxyref{External\-Event\-Timed\-Trace}{p.}{classdesmoj_1_1core_1_1simulator_1_1_external_event_timed_trace} class allows for inserting items with true time values into the trace file. When the event routine is called an appropriate trace note is sent. 

The {\ttfamily \doxyref{Experiment}{p.}{classdesmoj_1_1core_1_1simulator_1_1_experiment}} class uses {\ttfamily \doxyref{External\-Event\-Timed\-Trace}{p.}{classdesmoj_1_1core_1_1simulator_1_1_external_event_timed_trace}} events to write the start and stop time of the trace period as true time into the trace file.
