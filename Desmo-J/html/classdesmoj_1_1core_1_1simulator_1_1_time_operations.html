Detailed Description

TimeOperations is an utility class that provides arithmetic operations for the time classes TimeInstant and TimeSpan. It also holds the time settings, i.e. the granularity (epsilon) and the reference time unit.

Returns a new Time Instant determined from the instant passed to this method plus the time span passed to method. In other words, this method determines the instant that is a given span after the instant provided.
