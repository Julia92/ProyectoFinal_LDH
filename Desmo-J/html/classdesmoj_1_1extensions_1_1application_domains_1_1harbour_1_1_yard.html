Detailed Description

Yard is the place where containers (or other kind of goods) can be stored by transpoters or cranes and retrieved by transporters or cranes. The yard consists of blocks. The storing in the Yard is possible using store(Block b) .With the retrieve(Block b) a container can be retrieved from the Yard. Yard is part of the composite design pattern as described in [Gamm97] page 163. Yard is derived from Block, which provides the functionality of the Block.
