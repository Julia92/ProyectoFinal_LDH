Detailed Description

Empirically distributed stream of pseudo random numbers of type double. Values produced by this distribution follow an empirical distribution which is specified by entries consisting of the observed value and the frequency (probability) this value has been observed to occur. These entries are made by using the addEntry() method. There are a few conditions a user has to meet before actually being allowed to take a sample of this distribution.

void desmoj.core.dist.ContDistEmpirical.addEntry ( double  value,
double  frequency

Adds a new entry of an empirical value and its associated cumulative frequency. No two values may be equal, since there can not be two different frequencies for one observed value. Entries with same value/frequency pair as an entry already made before are simply ignored.

As soon as there is exactly one value with a cumulative probability of 0 and one value with a cumulative probability of 1, this distribution is considered "initialized", making it possible to obtain samples via the sample() method.

Parameters
valuedouble : The empirical value observed
frequencydouble : The corresponding cumulative frequency of the empirical value
