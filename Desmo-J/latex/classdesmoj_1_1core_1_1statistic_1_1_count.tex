\subsection{Detailed Description}
The {\ttfamily \doxyref{Count}{p.}{classdesmoj_1_1core_1_1statistic_1_1_count}} class is simply counting something during an experiment. Be careful the counter can be updated with positive and negative integer numbers!

To get this {\ttfamily \doxyref{Count}{p.}{classdesmoj_1_1core_1_1statistic_1_1_count}} object updated automatically every time a {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} has changed, call the {\ttfamily add\-Observer (Observer)} -\/method from the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} of interest, where Observer is this {\ttfamily \doxyref{Count}{p.}{classdesmoj_1_1core_1_1statistic_1_1_count}} object. \par
 This must be done by the user in his model! \par
 Consider usage of class {\ttfamily \doxyref{Aggregate}{p.}{classdesmoj_1_1core_1_1statistic_1_1_aggregate}} to count non-\/integer (i.\-e. double) values.
