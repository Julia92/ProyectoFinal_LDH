\subsection{Detailed Description}
Interface to restrict the functionalities of a parameter-\/manager to manage model-\/parameters. This Interface should be used as return-\/ type of the \doxyref{Model}{p.}{classdesmoj_1_1core_1_1simulator_1_1_model}'s methods which return the parameter-\/ manager.
