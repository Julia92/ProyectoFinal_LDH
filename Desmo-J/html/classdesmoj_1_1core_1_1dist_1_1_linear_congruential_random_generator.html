Detailed Description

Linear congruential random generator for uniformly distributed pseudo random numbers configured such that a stream of [0,1] double values is produced. Implements the desmoj.dist.UniformRandomGenerator interface. All Distributions in this package use this random generator by default. The implementation is based on the Java API java.util.Random class' random generator. The Java API Random class uses a 48-bit seed as input to the linear congruential formula. (See Donald Knuth, The Art of Computer Programming, Volume 2, Section 3.2.1.)
