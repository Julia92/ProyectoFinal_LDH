\subsection{Detailed Description}
Class to Represent a \doxyref{Parameter}{p.}{classdesmoj_1_1core_1_1simulator_1_1_parameter}. This parameter can be an experiment-\/ parameter or a model-\/parameter. Parameters are variables that are accessible from everywhere in the model. Model-\/parameters are assignable from model, experiment-\/parameters from experiment.
