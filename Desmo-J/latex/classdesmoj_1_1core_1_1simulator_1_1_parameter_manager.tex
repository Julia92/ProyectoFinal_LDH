\subsection{Detailed Description}
Class to manage parameters. Parameters are values, which affect a simulation-\/ model or an experiment. Model-\/parameters are model's constants. Experiment-\/ parameters are experiment-\/constants which can be used in the experiment's model.

This class is accessible from \doxyref{Model}{p.}{classdesmoj_1_1core_1_1simulator_1_1_model} through the Model\-Parameter\-Manager-\/interface to restrict the accessible methods from a model's view. It is accessible from \doxyref{Experiment}{p.}{classdesmoj_1_1core_1_1simulator_1_1_experiment} through the Experiment\-Parameter\-Manager-\/interface to restrict the accessible methods from a experiment's view.
