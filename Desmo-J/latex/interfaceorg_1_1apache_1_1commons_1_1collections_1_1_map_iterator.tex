\subsection{Detailed Description}
Defines an iterator that operates over a {\ttfamily Map}. 

This iterator is a special version designed for maps. It can be more efficient to use this rather than an entry set iterator where the option is available, and it is certainly more convenient. 

A map that provides this interface may not hold the data internally using Map Entry objects, thus this interface can avoid lots of object creation. 

In use, this iterator iterates through the keys in the map. After each call to {\ttfamily \doxyref{next()}{p.}{interfaceorg_1_1apache_1_1commons_1_1collections_1_1_map_iterator_a9c0eb3ac857fdfc06d0a6362f59ae718}}, the {\ttfamily \doxyref{get\-Value()}{p.}{interfaceorg_1_1apache_1_1commons_1_1collections_1_1_map_iterator_a1fa1975927c72e42cf45ea1cefa819bd}} method provides direct access to the value. The value can also be set using {\ttfamily \doxyref{set\-Value()}{p.}{interfaceorg_1_1apache_1_1commons_1_1collections_1_1_map_iterator_a9e6e507237a309918633cab0009426d1}}.
