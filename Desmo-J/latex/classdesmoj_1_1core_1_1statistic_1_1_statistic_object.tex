\subsection{Detailed Description}
{\ttfamily \doxyref{Statistic\-Object}{p.}{classdesmoj_1_1core_1_1statistic_1_1_statistic_object}} class is the super class of all other classes collecting statistical data. \par
 It extends the {\ttfamily \doxyref{desmoj.\-core.\-simulator.\-Reportable}{p.}{classdesmoj_1_1core_1_1simulator_1_1_reportable}} class so that it can provide a reporter to represent the statistical data in the report. \par
 It also implements the {\ttfamily java.\-util.\-Observer} interface. So this is the observer part of the observer pattern as described in [Gamm95] that is observing the {\ttfamily desmoj.\-statistic.\-Value\-Supplier}. Whenever the {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} is changing it calls the {\ttfamily update(\-Observable, Object)} method of this {\ttfamily \doxyref{Statistic\-Object}{p.}{classdesmoj_1_1core_1_1statistic_1_1_statistic_object}}. Which happens inside the {\ttfamily notify\-Observers()} method of the observable {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}}. 

The virtual method {\ttfamily update(\-Observable, Object)} has to be implemented in all derived classes so that the {\ttfamily \doxyref{Statistic\-Object}{p.}{classdesmoj_1_1core_1_1statistic_1_1_statistic_object}} is updated when the observed {\ttfamily \doxyref{Value\-Supplier}{p.}{classdesmoj_1_1core_1_1statistic_1_1_value_supplier}} has changed and its {\ttfamily notify\-Observers()} method is called from its {\ttfamily notify\-Statistics()} method.
