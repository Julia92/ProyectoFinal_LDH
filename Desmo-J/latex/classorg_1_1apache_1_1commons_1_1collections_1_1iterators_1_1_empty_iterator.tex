\subsection{Detailed Description}
Provides an implementation of an empty iterator. 

This class provides an implementation of an empty iterator. This class provides for binary compatability between Commons Collections 2.\-1.\-1 and 3.\-1 due to issues with {\ttfamily Iterator\-Utils}.
