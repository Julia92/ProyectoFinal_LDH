\subsection{Detailed Description}
Represents a polynomial spline function. 

A {\bfseries polynomial spline function} consists of a set of {\itshape interpolating polynomials} and an ascending array of domain {\itshape knot points}, determining the intervals over which the spline function is defined by the constituent polynomials. The polynomials are assumed to have been computed to match the values of another function at the knot points. The value consistency constraints are not currently enforced by {\ttfamily \doxyref{Polynomial\-Spline\-Function}{p.}{classorg_1_1apache_1_1commons_1_1math_1_1analysis_1_1_polynomial_spline_function}} itself, but are assumed to hold among the polynomials and knot points passed to the constructor.

N.\-B.\-: The polynomials in the {\ttfamily polynomials} property must be centered on the knot points to compute the spline function values. See below.

The domain of the polynomial spline function is {\ttfamily [smallest knot, largest knot]}. Attempts to evaluate the function at values outside of this range generate Illegal\-Argument\-Exceptions. 

The value of the polynomial spline function for an argument {\ttfamily x} is computed as follows\-: 
\begin{DoxyEnumerate}
\item The knot array is searched to find the segment to which {\ttfamily x} belongs. If {\ttfamily x} is less than the smallest knot point or greater than the largest one, an {\ttfamily Illegal\-Argument\-Exception} is thrown. 
\item Let {\ttfamily j} be the index of the largest knot point that is less than or equal to {\ttfamily x}. The value returned is \par
 {\ttfamily polynomials[j](x -\/ knot[j])}
\end{DoxyEnumerate}
