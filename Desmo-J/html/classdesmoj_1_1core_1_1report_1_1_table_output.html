Detailed Description

An output for tables containing simulation results. The table formatting is done by an assigned TableFormatter (e.g. a HTMLTableFormatter or an ASCIITableFormatter). Realizes the output functionality of the deprecated class desmoj.report.HTMLFileOut. This class is the new base class for the different Desmo-J output channels (ReportFileOut, etc.)
