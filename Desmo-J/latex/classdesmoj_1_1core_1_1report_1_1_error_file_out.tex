\subsection{Detailed Description}
\doxyref{Error\-File\-Out}{p.}{classdesmoj_1_1core_1_1report_1_1_error_file_out} is used to create a file to write the Error\-Messages to. It receives an \doxyref{Error\-Message}{p.}{classdesmoj_1_1core_1_1report_1_1_error_message} and divides its individual text information up to be displayed in H\-T\-M\-L format on disc in the user's local directory. The messages are displayed in a tabular design with columns for\-: 
\begin{DoxyItemize}
\item The point of simulation time the \doxyref{Error\-Message}{p.}{classdesmoj_1_1core_1_1report_1_1_error_message} was created. 
\item A textual description of what went wrong. 
\item The location that issued the \doxyref{Error\-Message}{p.}{classdesmoj_1_1core_1_1report_1_1_error_message}. 
\item The possible reason why that error condition occurred. 
\item A hint for the user on how to prevent this error to happen again. 
\end{DoxyItemize}Note that if the framework is used as an Applet in webpages there is no access to the disc and an alternative output on screen or into a graphics window must be registered at the Experiment's Message\-Manager for this type of messages. Errors affecting the java runtime are always displayed on the system's standard output Print\-Stream.
