Detailed Description

Is the class summing up all the collective implementation of different queueing strategies for a queue list. It provides all the basic methods for inserting objects in a queue, retrieving objects from a queue and getting basic informations about the queue. It is used in many kinds of queue implementations e.g. in the classes QueueListFifo and QueueListLifo.

abstract void desmoj.core.simulator.QueueListStandard< E extends Entity >.insert ( E  e)
pure virtual

Adds a new Entity to the QueueList. Entities are inserted according to their priority in descending order. The highest priority Entity will always be first in the queue. Entities with same priority are inserted in specified order.

Do not forget to call the statisticalInsert() and to set the queue for each entity as you define this method.

boolean desmoj.core.simulator.QueueListStandard< E extends Entity >.remove ( E  e)

Removes the first occurrence of the given Entity from the QueueList. Checks if the given Entity to be removed does apply to all restrictions on this operation. These are :

  • The given reference to an entity must not be null

  • This QueueList must not be empty, otherwise there's nothing to remove If all these restrictions apply, true is returned and the Entity is removed, otherwise false is the return value because the given Entity could not be removed since one of the restrictions above was not met.
