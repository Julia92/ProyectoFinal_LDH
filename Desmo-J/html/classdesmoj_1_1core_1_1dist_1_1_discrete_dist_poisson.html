Creates a poisson distributed stream of pseudo random integer numbers. The parameter describes the mean value. Giving zero or negative values for the mean value will result in zero being returned by the sample() method.

Parameters
ownerModel : The distribution's owner
namejava.lang.String : The distribution's name
meanValuedouble : The mean double value for this distribution
showInReportboolean : Flag for producing reports
showInTraceboolean : Flag for producing trace output
