Detailed Description

Captures all relevant information about the TransporterSystem. That means from the transporter and the jobs queue.

desmoj.extensions.applicationDomains.harbour.report.TSReporter.TSReporter ( desmoj.core.simulator.Reportable  informationSource)

Constructor for a new TSReporter. Note that although any Reportable is accepted you should make sure that only subtypes of TransporterSystem are passed to this constructor. Otherwise the number of column titles and their individual headings will differ from the actual content collected by this reporter.

String [] desmoj.extensions.applicationDomains.harbour.report.TSReporter.getEntries ( )
virtual

Returns an array of Strings each containing the data for the corresponding column in array columns[]. This method is fetching the data from this TransporterSystem. Implement this method in a way, that an array of the same length as the column titles is produced containing the data at the point of time this method is called by someone else to produce up-to-date information.
