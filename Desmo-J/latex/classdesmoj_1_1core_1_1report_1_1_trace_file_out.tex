\subsection{Detailed Description}
\doxyref{Trace\-File\-Out}{p.}{classdesmoj_1_1core_1_1report_1_1_trace_file_out} is used to create a file to write the tracemessages to. It receives a tracemessage and divides its individual text information up to be displayed in H\-T\-M\-L format on disc in the user's local directory. The messages are displayed in a tabular design with columns for\-: 
\begin{DoxyItemize}
\item The point of simulation time the tracemessage was created 
\item The entity's name responsible for sending the tracemessage or '-\/' in case of an external event 
\item The event associated with the entity or '-\/' in case of a Sim\-Process  
\item The textual description of what has happened to the model state 
\end{DoxyItemize}Note that if the framework is used as an applet in webpages there is no access to the disc and an alternative output on screen or into a graphics window must be registered at the experiment's messagemanager for this type of messages. Errors affecting the java runtime are always displayed on the system's standard output printstream.
