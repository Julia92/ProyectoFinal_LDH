\subsection{Detailed Description}
This factory provids the means to create common statistical distributions. The following distributions are supported\-: 
\begin{DoxyItemize}
\item Binomial 
\item Cauchy 
\item Chi-\/\-Squared 
\item Exponential 
\item F 
\item Gamma 
\item Hyper\-Geometric 
\item Poisson 
\item Normal 
\item Student's t 
\item Weibull 
\item Pascal 
\end{DoxyItemize}
