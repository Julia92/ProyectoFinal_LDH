Detailed Description

A CranesSystem represents the system of the cranes that manages the queues of the cranes and transporter (external/internal), give the statistics about the cranes and both queues.

Constructor for a CranesSystem. The queueing discipline and the capacity limit of the underlying queues for the cranes and for the transporters can be chosen.

Parameters
ownerModel : The model this CranesSystem is associated to.
namejava.lang.String : The CranesSystem's name
csortOrderint : determines the sort order of the underlying cranes queue implementation. Choose a constant from QueueBased like QueueBased.FIFO or QueueBased.LIFO or ...
cQCapacityint : The capacity of the cranes queue, that is how many cranes can be enqueued. Zero (0) means unlimited capacity.
tsortOrderint : determines the sort order of the underlying transporter queue implementation. Choose a constant from QueueBased like QueueBased.FIFO or QueueBased.LIFO or ...
tQCapacityint : The capacity of the transporter queue, that is how many transporters can be enqueued. Zero (0) means unlimited capacity.
showInReportboolean : Flag, if this CranesSystem should produce a report or not.
showInTraceboolean : Flag for trace to produce trace messages.
