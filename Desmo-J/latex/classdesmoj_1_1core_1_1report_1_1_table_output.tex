\subsection{Detailed Description}
An output for tables containing simulation results. The table formatting is done by an assigned \doxyref{Table\-Formatter}{p.}{interfacedesmoj_1_1core_1_1report_1_1_table_formatter} (e.\-g. a \doxyref{H\-T\-M\-L\-Table\-Formatter}{p.}{classdesmoj_1_1core_1_1report_1_1_h_t_m_l_table_formatter} or an \doxyref{A\-S\-C\-I\-I\-Table\-Formatter}{p.}{classdesmoj_1_1core_1_1report_1_1_a_s_c_i_i_table_formatter}). Realizes the output functionality of the deprecated class desmoj.\-report.\-H\-T\-M\-L\-File\-Out. This class is the new base class for the different Desmo-\/\-J output channels (\doxyref{Report\-File\-Out}{p.}{classdesmoj_1_1core_1_1report_1_1_report_file_out}, etc.)
