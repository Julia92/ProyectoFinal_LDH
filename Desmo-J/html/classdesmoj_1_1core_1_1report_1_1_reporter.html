Detailed Description

Declares the basic methods needed for reporter to be able to print reports about a model component. Since individual components need individual reports each component has to specify a special reporter to print its data. Derive from this class to implement the reporter for the specific Reportable class.

int desmoj.core.report.Reporter.groupID
protected

The group-id of the reporters of this class. This ID resembles group-identification and ordering information inside a group. The HTML output classes order Reporters by groups of hundred ID numbers. So i.e. all Distributions have Reporters with GroupID's in the range between 100 and 199. Highest numbers are always listed first. If in a sequence of Reporters its GroupID is in another century range, the HTML output will automatically insert a horizontal ruler and a new heading to indicate the new group of Reportable ModelComponents.

Reserved GroupID ranges
GroupID range Reportables
2147483647 ModelReporter. Highest number possible, so always first in order. Ensures that SubModelReporter will be printed before other Reportables in the main model report.
2000000000 SimulationrunReporter, including Experiment duration and other important information referring to the experiment as whole.
1800 + Free for additional Reporters of new constructs that will be listed before the StatisticObjects used in a Model.
1300 - 1799 Reporters for StatisticObjects as the Accumulate, Tally, TextHistogram, Histogram, Regression and Count, which will be listed before the process synchronisation constructs used in the Model.
400 - 1299 Additional Reporters for process synchronisation constructs as the Bin, Stock, Entrepot, Res, CondQueue, WaitQueue, Transporter and WorkStation, which will be listed before the Queues used in a Model.
300 - 399 Free for additional Reporters of new constructs that will be listed before the Queues used in a Model.
200 - 299 Queues. Queue will be sorted first, ProcessQueue last
100 - 199 Distributions. The generic DistributionReporter will be last with its GroupID of 100.
1-99 Free for additional Reporters of new constructs that will be listed after the Queues used in a Model.
