\subsection{Detailed Description}
This class contains all elements needed in a scene graph to describe a dynamic object (i.\-e. a Movable\-Spatial\-Object) in a scene.

A \doxyref{Dynamic\-Object}{p.}{classdesmoj_1_1extensions_1_1visualization3d_1_1_dynamic_object} contains several elements\-: 
\begin{DoxyItemize}
\item a Branch\-Group for attach/detach it to/from nodes
\item a Branch\-Group which contains the 3\-D-\/model
\item a Transform\-Group which's responsible for the positioning
\item a Transform\-Group which's responsible for the orientation
\item a \doxyref{Movement\-Path\-Interpolator}{p.}{classdesmoj_1_1extensions_1_1visualization3d_1_1_movement_path_interpolator} to move the object in time
\item a Rotation to rotate the object in time
\end{DoxyItemize}
