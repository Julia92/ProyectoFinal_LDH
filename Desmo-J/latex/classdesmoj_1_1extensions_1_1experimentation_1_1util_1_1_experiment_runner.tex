\subsection{Detailed Description}
A thread a Desmo\-J-\/\-Experiment can run in. The experiment runner can notifiy registered Experiment\-Listeners when the assigned experiment is (re)started, stopped or temporarily paused and registered Sim\-Clock\-Listeners when the simulation clock is set forth.
