void desmoj.core.statistic.Regression.update ( Observable  o,
Object  arg 
)

Implementation of the virtual update(Observable, Object) method of the Observer interface. This method will be called automatically from an Observable object within its notifyObservers() method.
Whenever either of the two ValueSupplier objects providing the x or y value is updated and calling notifyStatistics() the x and the y value will be fetched and a new regression will be calculated.

Parameters
ojava.util.Observable : The Observable calling this method within its own notifyObservers() method.
argObject : The Object with which this Regression is updated.
