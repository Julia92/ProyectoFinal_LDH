\subsection{Detailed Description}
\doxyref{Experiment}{p.}{classdesmoj_1_1core_1_1simulator_1_1_experiment} is the class that provides the infrastructure for running the simulation of a model. It contains all data structures necessary to simulate the model and takes care of all necessary output. To actually run an experiment, a new instance of the experiment class and a new instance of the desired model have to be created. To link both instances, call the {\ttfamily connect\-To\-Experiment(\-Experiment e)} method of the model instance and pass the new experiment as a parameter.
