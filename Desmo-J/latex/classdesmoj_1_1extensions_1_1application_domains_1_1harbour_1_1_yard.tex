\subsection{Detailed Description}
\doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard} is the place where containers (or other kind of goods) can be stored by transpoters or cranes and retrieved by transporters or cranes. The yard consists of blocks. The storing in the \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard} is possible using {\ttfamily store(\-Block b)} .With the {\ttfamily retrieve(\-Block b)} a container can be retrieved from the \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard}. \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard} is part of the composite design pattern as described in [Gamm97] page 163. \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard} is derived from \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block}, which provides the functionality of the \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block}.
