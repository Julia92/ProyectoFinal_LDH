\subsection{Detailed Description}
{\ttfamily \doxyref{Statistic\-Object\-Supporting\-Time\-Spans}{p.}{classdesmoj_1_1core_1_1statistic_1_1_statistic_object_supporting_time_spans}} class is the super class of the specific data collector classes able to handle {\ttfamily Time\-Span} objects. If the data collector should be used to measure the lengths of {\ttfamily Time\-Span}s, the method {\ttfamily set\-Show\-Time\-Spans\-In\-Report(true)} should be called to appropriately assign the (reference) unit.
