\subsection{Detailed Description}
All entities of same entity-\/type has the same basic properties. These are\-:
\begin{DoxyItemize}
\item same size in animation (width/heigth)
\item same possible states with same icon for a state
\item same possible attribute-\/key's
\end{DoxyItemize}
