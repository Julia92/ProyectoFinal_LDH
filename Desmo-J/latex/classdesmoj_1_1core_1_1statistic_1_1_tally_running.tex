\subsection{Detailed Description}
The {\ttfamily \doxyref{Tally\-Running}{p.}{classdesmoj_1_1core_1_1statistic_1_1_tally_running}} class is extended from {\ttfamily \doxyref{Tally}{p.}{classdesmoj_1_1core_1_1statistic_1_1_tally}}. Same as {\ttfamily \doxyref{Tally}{p.}{classdesmoj_1_1core_1_1statistic_1_1_tally}}, it provides statistic analysis about one value (mean, standard deviation, minimum, maximum). In addition, this class is able to calculate this data based on the last {\itshape n} samples only, subject to {\itshape n} passed to the constructor.
