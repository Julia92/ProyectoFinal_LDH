\subsection{Detailed Description}
\doxyref{Time\-Operations}{p.}{classdesmoj_1_1core_1_1simulator_1_1_time_operations} is an utility class that provides arithmetic operations for the time classes \doxyref{Time\-Instant}{p.}{classdesmoj_1_1core_1_1simulator_1_1_time_instant} and \doxyref{Time\-Span}{p.}{classdesmoj_1_1core_1_1simulator_1_1_time_span}. It also holds the time settings, i.\-e. the granularity (epsilon) and the reference time unit.
