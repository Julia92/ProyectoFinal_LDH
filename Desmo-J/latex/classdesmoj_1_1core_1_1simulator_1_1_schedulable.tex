\subsection{Detailed Description}
All objects that want to be handled by the scheduler must extend this class. These are events (including external events), entities and Sim\-Processes. All functionality common to \doxyref{Schedulable}{p.}{classdesmoj_1_1core_1_1simulator_1_1_schedulable} objects are encapsulated here.
