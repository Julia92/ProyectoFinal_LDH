\subsection{Detailed Description}
Boolean Bernoulli distribution returning {\ttfamily true} values with the given probability. Samples of this distribution can either be true or false with a given probability for value \char`\"{}true\char`\"{}. The probabilitiy for \char`\"{}true\char`\"{} can only be set via the constructor. It has to be a value between 0 and 1. Higher values will be interpreted as 1 (always return \char`\"{}true\char`\"{}). Negative values will be interpreted as 0 (always return \char`\"{}false\char`\"{}).
