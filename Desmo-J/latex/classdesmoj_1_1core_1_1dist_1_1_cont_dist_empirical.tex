\subsection{Detailed Description}
Empirically distributed stream of pseudo random numbers of type {\ttfamily double}. Values produced by this distribution follow an empirical distribution which is specified by entries consisting of the observed value and the frequency (probability) this value has been observed to occur. These entries are made by using the {\ttfamily \doxyref{add\-Entry()}{p.}{classdesmoj_1_1core_1_1dist_1_1_cont_dist_empirical_a5f736f32e180e82d2ac50b3815a65fca}} method. There are a few conditions a user has to meet before actually being allowed to take a sample of this distribution.

Adds a new entry of an empirical value and its associated cumulative frequency. No two values may be equal, since there can not be two different frequencies for one observed value. Entries with same value/frequency pair as an entry already made before are simply ignored.

As soon as there is exactly one value with a cumulative probability of 0 and one value with a cumulative probability of 1, this distribution is considered \char`\"{}initialized\char`\"{}, making it possible to obtain samples via the \doxyref{sample()}{p.}{classdesmoj_1_1core_1_1dist_1_1_cont_dist_empirical_ac0fb6262033e3dc62784c17245cf2393} method.
