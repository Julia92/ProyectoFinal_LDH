\subsection{Detailed Description}
Implementation of D\-E\-S\-M\-O-\/\-J \doxyref{Output\-Type}{p.}{interfacedesmoj_1_1core_1_1report_1_1_output_type} which act similar to unix /dev/null simmply ignoring all received messaged and reporters. It also doesn't create any system resource. This implementation can be used if you want to suppress creation of default file(s) with experiment output.
