\subsection{Detailed Description}
Normally (also known as \char`\"{}\-Gaussian\char`\"{}) distributed stream of pseudo random numbers of type double. The algorithm used here is derived from the Java A\-P\-I class {\ttfamily java.\-util.\-Random} and modified to also produce antithetic values if antithetic mode is switched on.

Returns the next normally (also known as \char`\"{}\-Gaussian\char`\"{}) distributed sample from this distribution. The value depends upon the seed, the number of values taken from the stream by using this method before and the mean and standard deviation values specified for this distribution. Computation same as Real\-Dist\-Gaussian plus cast to {\ttfamily long}. The basic algorithm has been taken from the Java A\-P\-I {\ttfamily java.\-util.\-Random.\-next\-Gaussian()} with the feature of antithetic random numbers added.

Flag indicates wether there is already a calculated Gaussian value present in the buffer variable {\ttfamily next\-Gaussian}. Necessary for algorithm taken from Java A\-P\-I class {\ttfamily java.\-util.\-Random}. When calculating a pseudo random number using the algorithm implemented here, two Gaussian values are computed at a time, so the other value is stored to be used next time the client asks for a new value, thus saving on computation time.
