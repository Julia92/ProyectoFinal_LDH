\subsection{Detailed Description}
Represents a \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} produced by a simulation run whenever the modeller enables the debug mode for a specific Model\-Component. Provides the basic information needed to debug a model's changes of state\-: 
\begin{DoxyItemize}
\item The Model this \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} originates from 
\item The point in simulation time that this \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} was issued 
\item The name of the Model\-Component that produced the \doxyref{Debug\-Note}{p.}{classdesmoj_1_1core_1_1report_1_1_debug_note} 
\item The textual debug information produced by that Model\-Component 
\end{DoxyItemize}
