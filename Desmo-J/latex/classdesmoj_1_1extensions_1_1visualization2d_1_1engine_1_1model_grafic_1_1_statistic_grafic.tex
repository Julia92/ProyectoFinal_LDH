\subsection{Detailed Description}
Class to animate a instance of class Statistic. For animation you can choose a default animation type\-: A\-N\-I\-M\-A\-T\-I\-O\-N\-\_\-\-Last\-Value, A\-N\-I\-M\-A\-T\-I\-O\-N\-\_\-\-Time\-Value\-Diagram, A\-N\-I\-M\-A\-T\-I\-O\-N\-\_\-\-Histogram. Later, you can change interactively, with a popup menu, between the animation types.

For A\-N\-I\-M\-A\-T\-I\-O\-N\-\_\-\-Time\-Value\-Diagram and A\-N\-I\-M\-A\-T\-I\-O\-N\-\_\-\-Histogram the j\-Free\-Chart Library is used.
