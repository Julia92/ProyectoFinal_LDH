\subsection{Detailed Description}
A \doxyref{Choose\-Block\-Yard\-Strategy}{p.}{interfacedesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_choose_block_yard_strategy} is an interface and presents the strategy that is used by the {\ttfamily \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard}} to find a certain block to store or retrieve a container there. The method \doxyref{get\-Block(\-Block[$\,$] blocks)}{p.}{interfacedesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_choose_block_yard_strategy_a7bc71bcb845b45dbcc7bd7faca5b8fba} that gives the needed block must be implemented by the user in a class that implements \doxyref{Transport\-Strategy}{p.}{interfacedesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_transport_strategy} to define the strategy which will be used by the yard. \doxyref{Choose\-Block\-Yard\-Strategy}{p.}{interfacedesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_choose_block_yard_strategy} is part of the strategy design pattern as described in [Gamm97] page 333 in which it represents the strategy interface.
