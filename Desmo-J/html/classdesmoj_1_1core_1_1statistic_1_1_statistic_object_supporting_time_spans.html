Detailed Description

StatisticObjectSupportingTimeSpans class is the super class of the specific data collector classes able to handle TimeSpan objects. If the data collector should be used to measure the lengths of TimeSpans, the method setShowTimeSpansInReport(true) should be called to appropriately assign the (reference) unit.
