Detailed Description

Class to plot DesmoJ histogram, histogramAccumulater and time-series data in jFreeChart Plotter. When in DesmoJ dataset getShowTimeSpansInReport() is set, the data values are interpreted as a timespan in a appropriate time unit. The DesmoJ datasets are converted in jFreeChart Format. Onscreen and offscreen plots are supported.

See also PaintPanel and TimeSeriesDataSetAdapter
