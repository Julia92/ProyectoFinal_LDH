Creates a stream of pseudo random numbers following a Beta distribution. The specific parameters alpha and beta have to be given here at creation time.

Parameters
ownerModel : The distribution's owner
namejava.lang.String : The distribution's name
alphadouble: Distribution specific parameter alpha
betadouble : Distribution specific parameter beta
showInReportboolean : Flag for producing reports
showInTraceboolean : Flag for producing trace output
