\subsection{Detailed Description}
Derive from this class to create a conditional which the model as whole may fulfill or not, e.\-g. for the purpose of conditionally stopping an experiment. Override method check(\-E e) such that it returns {\ttfamily true} whenever the model complies to the condition.
