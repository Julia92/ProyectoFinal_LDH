\subsection{Detailed Description}
\doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block} is the place where containers (or other kind of goods) can be stored by transpoters or cranes and retrieved by transporters or cranes. The storing is possible using {\ttfamily \doxyref{store()}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block_ada41f51aa660ecb430a0a22b82988f03}} of the \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block}. With the {\ttfamily \doxyref{retrieve()}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block_ac3004e549c1ca766076ba488732cab86}} a container can be retrieved from the \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block}. A \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block} is a part of the container {\ttfamily \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard}}. There are no queues for processes want to store or retrieve one container. If no or not enough containers are available to retrieve, the tries to do that will be refused. The \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block} has a certain capacity that measures in T\-E\-Us. So a 20-\/foot-\/container weigths one T\-E\-U and 40-\/foot-\/container 2 T\-E\-Us. If the \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block} is filled to it's capacity the tries to store a container there will be refused. \doxyref{Yard}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_yard} is part of the composite design pattern as described in [Gamm97] page 163 in which it represents the the component class. \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block} is derived from Reportable, which provides the report functionality for the \doxyref{Block}{p.}{classdesmoj_1_1extensions_1_1application_domains_1_1harbour_1_1_block}.
