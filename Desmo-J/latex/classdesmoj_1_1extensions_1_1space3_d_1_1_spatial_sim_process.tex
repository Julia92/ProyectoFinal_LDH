\subsection{Detailed Description}
This class represents the D\-E\-S\-M\-O-\/\-J Sim\-Process which should be visualized but not movable in the 3\-D space. It implements the \doxyref{Spatial\-Object}{p.}{interfacedesmoj_1_1extensions_1_1space3_d_1_1_spatial_object} interface to handle with the 3\-D spatial data.
