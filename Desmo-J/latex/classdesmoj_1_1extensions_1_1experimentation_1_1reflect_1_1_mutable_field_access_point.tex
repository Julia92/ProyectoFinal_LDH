\subsection{Detailed Description}
A mutable access point using reflection to get and set a field's value.

This pattern can be found in several frameworks for agent-\/ or component-\/based simulation (e.\-g. 'Probes' in Swarm -\/ see www.\-swarm.\-org). The term 'access point' was adopted from the dissertation \char`\"{}\-Ein flexibler, 
\-C\-O\-R\-B\-A-\/basierter Ansatz fuer die verteilte, komponentenorientierte Simulation\char`\"{} by Ralf Bachmann (2003).
