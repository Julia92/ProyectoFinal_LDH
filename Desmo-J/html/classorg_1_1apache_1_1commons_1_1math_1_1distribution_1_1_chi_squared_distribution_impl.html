double org.apache.commons.math.distribution.ChiSquaredDistributionImpl.inverseCumulativeProbability ( final double  p) throws MathException

For this distribution, X, this method returns the critical point x, such that P(X < x) = p.

Returns 0 for p=0 and Double.POSITIVE_INFINITY for p=1.

Parameters

Returns
