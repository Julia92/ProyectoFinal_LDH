Detailed Description

A SpatialMovementManager is (only) associated to every movable descendant of MovableSpatialEntity and MovableSpatialSimProcess. It serves the interaction between the SpatialObjects, the IVisualObject, and the simulation model. It gets tracks from the model,calculates the duration, holds it self back during the animation and updates the coordinates to the SpatialObjects.

void desmoj.extensions.space3D.SpatialMovementManager.move ( SpatialObject  destination,

TimeSpan  simDuration,
double  maxSpeed,
double  acc,
double  dec 
)
protected

Moves the associated MovableSpatialObject to the position of an other SpatialObject with the given duration, maximum speed, acceleration and deceleration. The character of the movement depends on the initialized parameters. (O = initialized, X = not initialized(means <=0 or null)):

case 1: duration O, maxSpeed O, acc & dec O: The movement will be performed according to the parameters. If the given parameters can't be satisfied, an IllegalMoveException will be thrown.

case 2: duration X, maxSpeed O, acc & dec O: The object will be moved to the destination with the fastest speed possible.

case 3: duration O, maxSpeed X, acc & dec O: The object will move to the destination with the given acceleration and deceleration and try to reach the destination in time. If the given parameters can't be satisfied, an IllegalMoveException will be thrown.

case 4: duration X, maxSpeed X, acc & dec O: The object will be accelerated and followed by a decdleration. There's no constant phase in the movement.

case 5: duration O, maxSpeed O, acc or dec X: The object will start with the speed needed to arrive the destination in the given time and break suddenly to zero at the end of the movement. The maxSpeed gives the speed limit of the movement. If the given parameters can't be satisfied, an IllegalMoveException will be thrown.

case 6: duration X, maxSpeed O, acc or dec X: The movement will start with the maxSpeed and stops the destination.

case 7: duration O, maxSpeed X, acc or dec X: The movement will start with the speed needed to arrive the destination with the duration given.

Parameters
destinationThe destination of the movement.
enttryPointNameThe name of the specific entry point at the destination object, which should be steered to. (optional)
simDurationThe duration of the movement. (optional)
maxSpeedThe specific maximum speed for this movement. (optional)
accThe specific acceleration value for this movement. (optional)
decThe specific deceleration value for this movement. (optional)
