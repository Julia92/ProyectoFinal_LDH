\subsection{Detailed Description}
Interface for uniform pseudo random number generators to be used by the distribution classes to compute their samples. Random generators of this type must return uniform distributed double values. Note that two constructors have to be implemented. One constructor without parameters to construct a \doxyref{Uniform\-Random\-Generator}{p.}{interfacedesmoj_1_1core_1_1dist_1_1_uniform_random_generator} with seed zero, and another constructor with a parameter of type long to pass the initial seed value to the pseudo random generation algorithm.
