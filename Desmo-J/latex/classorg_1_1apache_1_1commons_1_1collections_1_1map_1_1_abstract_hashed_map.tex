\subsection{Detailed Description}
An abstract implementation of a hash-\/based map which provides numerous points for subclasses to override. 

This class implements all the features necessary for a subclass hash-\/based map. Key-\/value entries are stored in instances of the {\ttfamily Hash\-Entry} class, which can be overridden and replaced. The iterators can similarly be replaced, without the need to replace the Key\-Set, Entry\-Set and Values view classes. 

Overridable methods are provided to change the default hashing behaviour, and to change how entries are added to and removed from the map. Hopefully, all you need for unusual subclasses is here. 

N\-O\-T\-E\-: From Commons Collections 3.\-1 this class extends Abstract\-Map. This is to provide backwards compatibility for \doxyref{Reference\-Map}{p.}{classorg_1_1apache_1_1commons_1_1collections_1_1map_1_1_reference_map} between v3.\-0 and v3.\-1. This extends clause will be removed in v4.\-0.
